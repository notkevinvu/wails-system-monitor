# System Monitor Overlay - Features

## MVP Features (Phase 1)

### Core Metrics Display
- [ ] CPU usage percentage (overall)
- [ ] Memory usage (used/total, percentage)
- [ ] Real-time line chart visualization

### Overlay Window
- [ ] Frameless window
- [ ] Always-on-top option
- [ ] Draggable via custom handle
- [ ] Semi-transparent background

### Basic Controls
- [ ] Start/stop monitoring
- [ ] Close application

---

## Phase 2 Features

### Extended Metrics
- [ ] Per-core CPU usage
- [ ] Disk usage (read/write speed or capacity)
- [ ] Network traffic (upload/download speed)

### User Preferences
- [ ] Configurable update interval (500ms - 5s)
- [ ] Select which metrics to display
- [ ] Adjust window opacity
- [ ] Save/restore window position

### UI Improvements
- [ ] Multiple chart types (line, area, bar)
- [ ] Color themes (light/dark)
- [ ] Minimize to system tray

---

## Phase 3 Features (Future)

### Advanced
- [ ] Process list (top CPU/memory consumers)
- [ ] Alerts (threshold notifications)
- [ ] History export (CSV)
- [ ] Multiple monitor support

### System Tray
- [ ] Tray icon with quick stats tooltip
- [ ] Right-click menu
- [ ] Start minimized option

### Polish
- [ ] Keyboard shortcuts
- [ ] Auto-start with system
- [ ] Update checker

---

## Non-Goals (Out of Scope)

- Remote monitoring / multi-machine
- Historical data persistence (database)
- GPU monitoring (requires vendor-specific APIs)
- Mobile companion app
